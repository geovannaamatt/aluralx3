O que este design inclui: Paleta Gourmet Harmónica: Utiliza variáveis CSS (:root) para controlar as cores de forma centralizada. Tons inspirados em ingredientes reais que abrem o apetite. Layout Responsivo: Adapta-se automaticamente a ecrãs de telemóvel, tablet ou computador. Efeitos Visuais: Os blocos de pratos possuem uma sombra muito suave que se destaca ligeiramente (hover) quando passa o rato por cima. Fácil Customização: Para mudar o preço, o nome do prato ou as cores, basta alterar os textos diretamente no HTML ou os códigos hexadecimais no início do estilo.
